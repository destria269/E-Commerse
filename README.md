# UAS_E-Commerce
Dalam pembuatan Tugas UAS ini dengan menggunakan Laravel 
Laravel adalah sebuah framework PHP yang dirilis dibawah lisensi MIT, dibangun dengan konsep MVC (model view controller). Laravel adalah pengembangan website berbasis MVP yang ditulis dalam PHP yang dirancang untuk meningkatkan kualitas perangkat lunak dengan mengurangi biaya pengembangan awal dan biaya pemeliharaan, dan untuk meningkatkan pengalaman bekerja dengan aplikasi dengan menyediakan sintaks yang ekspresif, jelas dan menghemat waktu.

MVC adalah sebuah pendekatan perangkat lunak yang memisahkan aplikasi logika dari presentasi. MVC memisahkan aplikasi berdasarkan komponen- komponen aplikasi, seperti : manipulasi data, controller, dan user interface.

Model, Model mewakili struktur data. Biasanya model berisi fungsi-fungsi yang membantu seseorang dalam pengelolaan basis data seperti memasukkan data ke basis data, pembaruan data dan lain-lain.
View, View adalah bagian yang mengatur tampilan ke pengguna. Bisa dikatakan berupa halaman web.
Controller, Controller merupakan bagian yang menjembatani model dan view.


Pertama-tama saya membuat 3 tabel yaitu tabel kelas, tabel siswa, dan tabel user. Kemudian membuat CRUD .Lalu membuat tabel dengan menggunakan migration dan menampilkan data di views. Kemudian Create datadengan menambahkan feedback sukses atau gagal dan validasi form. kemudian mengupdate dan menghapus data. Kemudian membuat login untuk admin agar masuk pada tampilan home. dan bisa upload file berupa file gambar. 
Kemudian Markdown adalah sebuah tools untuk konversi text-to-HTML yang bisa digunakan ketika membuat konten pada web. Dengan Markdown memungkinkan kita untuk membuat sebuah format html dengan sangat sederhana dan ramah lingkungan (easy-to-read dan easy-to write). Kelebihan menggunakan Markdown adalah format penulisannya yang sangat manusiawi sehingga sangat mudah sekali untuk dibaca dan ditulis. Dengannya pula kita tidak perlu lagi dipusingkan dengan styling ketika menulis konten dan bisa fokus pada konten.
